# Processes and signals

| Filename | Description |
| -------- | ----------- |
| 0-what-is-my-pid | Bash displays its own PID |
| 1-list_your_processes | Displays a list of running processes |
| 2-show_your_bash_pid | Displays lines containing the word bash: (by 1-list_your_processes) |
| 3-show_your_bash_pid_made_easy | Displays the PID, with the process name that contains the word bash |
| 4-to_infinity_and_beyond | Displays To infinity and beyond indefinitely |
| 5-dont_stop_me_now | Stops 4-to_infinity_and_beyond loop |
| 6-stop_me_if_you_can | Stops 4-to_infinity_and_beyond loop |
| 7-highlander | Displays "To inifity and beyond" or "I am invincable!!!" when receiving a SIGTERM signal |
| 67-stop_me_if_you_can | Stops 7-highlander loop |
| 8-beheaded_process | Kills 7-highlander loop |
